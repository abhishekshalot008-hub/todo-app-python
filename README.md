# To-Do App (Python)

A command-line to-do list application built in Python.

## Features
- Add, view, mark complete, and delete tasks
- Persistent storage using JSON — tasks are saved between sessions

## How to run
python todolist.py

## What I learned
Built as my first full Python project — practiced loops, conditionals, 
functions, dictionaries, and file handling (JSON) end-to-end.
